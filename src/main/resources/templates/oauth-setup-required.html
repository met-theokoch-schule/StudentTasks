Welches IServ wird benutzt?

Der OAuth-Provider wird über ISERV_BASE_URL festgelegt. Standardmäßig ist aktuell https://theokoch.schule eingetragen.

Wenn ein anderes IServ verwendet werden soll, setze ISERV_BASE_URL auf die Basisadresse des Servers, zum Beispiel https://schule.example ohne abschließenden Slash.

Kurzanleitung für Ubuntu mit Tomcat
  1. Systemd-Override für den Tomcat-Service anlegen. Meist heißt der Dienst tomcat10, ggf. tomcat9.
  2. Die Umgebungsvariablen ISERV_BASE_URL, ISERV_CLIENT_ID und ISERV_CLIENT_SECRET dort setzen.
  3. Tomcat neu laden und neu starten.
sudo systemctl edit tomcat10

[Service]
Environment="ISERV_BASE_URL=[[${providerBaseUrlExample}]]"
Environment="ISERV_CLIENT_ID=deine-client-id"
Environment="ISERV_CLIENT_SECRET=dein-client-secret"
Environment="ISERV_REDIRECT_URI=[[${redirectUriExample}]]"

sudo systemctl daemon-reload
sudo systemctl restart tomcat10
