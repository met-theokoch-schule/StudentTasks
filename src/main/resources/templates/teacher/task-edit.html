Aufgabentypen

Verschiedene Aufgabentypen ermöglichen unterschiedliche Bearbeitungsformen:

  • Einfacher Texteditor: Für reine Textabgaben
  • HTML Editor: Für formatierte Texte
Markdown-Syntax

Die Aufgabenbeschreibung unterstützt Markdown:

  • **fett** für fett
  • *kursiv* für kursiv
  • # Überschrift für Überschriften
  • - Punkt für Listen
Standard-Vorlage

Mit der Standard-Vorlage können Sie den Schülern eine Ausgangsbasis geben. Das Format hängt vom gewählten Aufgabentyp ab.

Tutorial

Das Tutorial-Feld ermöglicht es, den Schülern zusätzliche Hilfestellungen oder Schritt-für-Schritt-Anleitungen zu geben. Das Format hängt vom gewählten Aufgabentyp ab.
