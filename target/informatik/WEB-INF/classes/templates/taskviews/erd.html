In der Schulbibliothek „Leseratte“ soll die Ausleihe künftig digital verwaltet werden. Jede **Schülerin** und jeder **Schüler** erhält dazu eine eindeutige **Schülernummer**; außerdem werden **Vorname**, **Nachname**, **Klasse** und eine **E-Mail-Adresse** gespeichert. Die Bibliothek besitzt **Bücher**, zu denen eine eindeutige **Buchnummer**, der **Titel**, das **Erscheinungsjahr** und die **ISBN** erfasst werden. Ein Buch kann von der Bibliothek in mehreren **Exemplaren** vorhanden sein. Jedes Exemplar hat eine eigene eindeutige **Exemplarnummer**, ein **Anschaffungsdatum** und einen **Zustand** (z. B. „neu“, „gut“, „beschädigt“). Jedes Exemplar gehört genau zu einem Buch; ein Buch kann entsprechend kein, ein oder mehrere Exemplare besitzen. Zu jedem Buch werden außerdem die **Autorinnen und Autoren** dokumentiert. Für Autorinnen/Autoren gibt es eine eindeutige **Autorennummer**, den **Namen** sowie das **Geburtsjahr**. Ein Buch kann von **mehreren** Autorinnen/Autoren geschrieben worden sein, und eine Autorin/ein Autor kann **mehrere** Bücher verfasst haben. Wenn ein Exemplar ausgeliehen wird, wird ein **Ausleihvorgang** gespeichert: Zu jeder Ausleihe werden das **Ausleihdatum**, das **Fälligkeitsdatum** und – falls das Exemplar zurückgegeben wurde – das **Rückgabedatum** erfasst. Eine Ausleihe bezieht sich immer auf **genau ein Exemplar** und wird immer von **genau einer Schülerin/einem Schüler** durchgeführt. Ein Exemplar kann im Laufe der Zeit **oft** ausgeliehen werden, aber **nicht gleichzeitig** in zwei laufenden Ausleihen stecken: Solange kein Rückgabedatum eingetragen ist, gilt das Exemplar als „ausgeliehen“ und kann nicht erneut ausgeliehen werden. Eine Schülerin/ein Schüler kann gleichzeitig mehrere Exemplare ausleihen, jedoch höchstens **drei** zur gleichen Zeit. Zusätzlich verwaltet die Bibliothek **Genres** (z. B. „Fantasy“, „Krimi“, „Sachbuch“). Jedes Genre hat eine eindeutige **Genre-ID** und eine **Bezeichnung**. Jedes Buch gehört **mindestens einem** Genre an, kann aber auch **mehreren** Genres zugeordnet sein. **Arbeitsauftrag:** Erstellt aus diesem Text ein **ER-Diagramm**.
